# App Development

## Project Overview
This project is developed as part of the Multidisciplinary Project Module (Project #14). It involves the creation of a mobile and/or web application aimed at addressing a real-world issue. The application integrates multiple technologies and development practices to ensure scalability, functionality, and an excellent user experience.

## Technologies Used
- **Frontend:**  
  - HTML  
  - CSS  
  - JavaScript  
  - React.js  

- **Backend:**  
  - JavaScript  
  - Node.js  
  - Express.js  

- **Database:**  
  - MySQL  

- **Version Control:**  
  - Git & GitHub  

## Key Features
- Secure user authentication and role-based authorization  
- Interactive and responsive UI/UX design  
- Persistent data storage with MySQL integration  
- RESTful APIs for seamless data communication  
- Real-time updates and notifications  

## Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   
2. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   npm run dev

3. **Backend Setup**
   ```bash
   cd backend
   npm install
   node index.js

4. **Database Setup**
- Ensure MySQL is installed and running.
- Create the required database and tables using the provided SQL script (if available).
- Configure database credentials in the backend (e.g., .env file).

## Usage
- Access the app via ```http://localhost:5173/``` after starting both frontend and backend.
- Register and log in to explore the application's features.
